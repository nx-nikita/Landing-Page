## 1- Clone the Repository:-
-``git clone https://github.com/nx-nikita/Landing-Page.git``

## 2 -Navigate into the project folder:-
``-cd Landing-Page``

## 3- **Prerequisites**
- bash->
-``node -v``
-``npm -v``

## 4- Installation:-
- ``npm install``
- 
## 5- Run:-
- ``npm start``
  
## 6- Open your browser and go to:
-``http://localhost:3000``

## 8- Key Highlights:-
- Fully **responsive design** that works on all devices.
- Uses **modern CSS features** like gradients, shadows, transitions, hover animations, and keyframe animations.
- Built with **React.js**, making it easy to extend or integrate with backend APIs.
- Interactive elements such as **play button overlays, scrollable playlists, and animated CTA** for a modern web experience.

## 9-Technologies Used:-
- **React.js** – Frontend library
- **CSS3 / Flexbox / Grid** – Layouts and styling
- **JavaScript (ES6+)** – Interactivity
- **Unsplash Images** – High-quality free images for playlists and hero section
- **Responsive Design** – Works on all screen sizes
